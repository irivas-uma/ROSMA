BOTONES:

- New user: crea un nuevo usuario, que sera Tx o Sx, dependiendo de si es un tecnico (T) o un cirujano (S). La x es el ID de la persona, que irá de uno en adelante. Cada persona deberá crear un perfil, con su nombre, género y edad. 
	* El perfil de cada usuario se guardará en un fichero txt dentro de una carpeta llamada profiles.
	
- Start: para comenzar con nuevo experimento. Se deberá de haber seleccionado la tarea y automaticamente aparecerá el trial como un display, para que podamos llevar la cuenta. 
	* Este botón se podrá activar, bien presionándolo, o bien se activará automáticamente cuando se presiona el pedal de coag.
	* Una vez se presione start, se manda un topic con el filename de ese trial concreto, en: /rosma/gui/filename (tipo string).
	* Se empezará a contar el tiempo, y además se deben lanzar dos rosbag record, uno para tener un bag de seguridad, que tendrá el mismo nombre que los csv pero con extension, y uno con las imágenes. El tiempo se guardára en un fichero a parte csv donde vengan todas las tareas de todos los usuarios y la duración de cada uno. Si se puede sensorizar el experimento, en este mismo fichero se pondrían los penalties y los scores.   
- Stop: se para el recording, escribiendo en el topic: /rosma/gui/stop_recording (tipo bool). 

- Repeat recording: borra el último fichero creado. 

- Fichero que vaya llevando la cuenta del numero de usuarios y cuantas repeticiones han hecho de cada experimento. 'dvrk_experiment_info.csv';
